# Arrays-Example
```JS
tasks = ['Code', 'Eat', 'Code'];
// ["Code", "Eat", "Code"]
```


# Sets Example

```JavaScript
tasks = new Set(['Code', 'Eat', 'Code']);
// ("Code", "Eat")
```


# Objects Example

```JavaScript
task = {
	task: 'Code',
	date: 'Today',
	repeat: 'True'
};
```


# Maps Example

```JavaScript
task = new Map ({
	['task', 'Code'],
	['date', 'today'],
	[false, 'Start coding!']
});
```
